# StefARR V0.6 rebrand

V0.6 introduces the public product name **StefARR by PaintedClouds** and an
original StefARR visual identity. The application remains an independent,
MIT-licensed fork of [Seerr](https://github.com/seerr-team/seerr). The upstream
copyright and permission notice remain unchanged in [LICENSE](../LICENSE), and
the exact source lineage is recorded in [ATTRIBUTION.md](../ATTRIBUTION.md).

## Upgrade behavior

- Existing default titles `Seerr` and `PaintedClouds Hub` are migrated once to
  `StefARR by PaintedClouds`.
- Administrator-defined instance titles and email sender names are preserved.
- The existing configuration directory, database schema, `/api/v1/hub` routes,
  `HUB_*` compatibility settings, container repository, and Compose/Helm
  resource identifiers remain stable in this release.
- PWA assets use new StefARR filenames to avoid stale operating-system icon
  caches. Existing asset names remain available for compatibility.

## Distribution and provenance

The repository and GHCR package retain their current technical names during the
compatibility migration window. Every release records the exact StefARR source commit
and upstream Seerr base, ships an SBOM, and preserves the original MIT notice.
StefARR is not affiliated with, endorsed by, or supported by Seerr or its
maintainers.

Back up the complete configuration directory and database before upgrading.
Downgrades require restoring the matching pre-upgrade backup.
